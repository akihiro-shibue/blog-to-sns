Setup Manual

blog-to-sns セットアップマニュアル

ブログURLを貼るだけで X・Facebook・note に投稿できる環境を作る

所要時間：約30分 必要スキル：ターミナルの基本操作 費用：Gemini APIのみ（実質無料）

このツールでできること

📝

投稿文の自動生成

X・Facebook・note用に文体を自動で変えて生成

🖼️

アイキャッチ画像生成

各SNSの推奨サイズで専用画像を自動生成

🚀

ワンクリック投稿

生成後にボタン1つで各SNSへ自動投稿

Setup Steps
1

Node.js をインストールする

まだインストールしていない場合は公式サイトからダウンロードします。

https://nodejs.org/ja/ から「LTS版（推奨版）」をダウンロード

インストール後、ターミナルで確認：

node -v
# v18.0.0 以上が表示されればOK
2

GitHubからファイルをダウンロードする

ターミナルを開いて以下を実行します。

git clone https://github.com/akihiroshibue/blog-to-sns.git
cd blog-to-sns
Gitが入っていない場合は https://git-scm.com/ からインストールしてください。
3

必要なパッケージをインストールする

blog-to-sns フォルダの中で以下を実行します。

npm install
npx playwright install chromium
# chromiumのインストールに1〜2分かかります
4

ブリッジサーバーを起動する

自動投稿に必要なサーバーを起動します。ターミナルを開いたまま別タブで実行します。

node bridge-server.mjs
# 「Bridge server listening on port 3004」と表示されればOK
このターミナルは閉じないでください。使い終わったら Ctrl+C で停止できます。
5

index.html をブラウザで開く

blog-to-sns フォルダ内の index.html をダブルクリックするか、以下で開きます。

open index.html # Mac の場合
start index.html # Windows の場合
6

APIキーを設定する

アプリを開いたら「⚙️ 設定」タブから各種キーを入力します。次のセクションで取得方法を説明します。

API Key Setup

設定タブで入力するキーの取得方法です。Gemini APIキーだけあれば投稿文生成はできます。

Gemini

Gemini API キー 必須

  1. Google AI Studio を開く
  2. Googleアカウントでログイン
  3. 「Create API key」をクリック
  4. 生成されたキーをコピー
無料枠で月数百回は使えます
X (Twitter)

X API キー 自動投稿時のみ

  1. X Developer Portal を開く
  2. アプリを作成（Basic プランで可）
  3. 「Keys and Tokens」から4つのキーを取得：
    API Key / API Secret /
    Access Token / Access Token Secret
Facebook

Facebook アクセストークン 自動投稿時のみ

  1. Meta for Developers を開く
  2. アプリを作成
  3. Graph API エクスプローラーで
    ページのアクセストークンを取得
  4. ページIDもコピーしておく
note

noteアカウント情報 自動投稿時のみ

  1. noteのメールアドレスとパスワードをそのまま設定タブに入力
  2. Playwrightが自動でログインして下書き保存します
パスワードはlocalStorageに保存されます（外部送信なし）
How to Use
1

「記事取得」タブでブログのURLを入力

URLを貼り付けるか、テキストを直接入力します

↓
2

「生成」タブで「一括生成」ボタンを押す

X・Facebook・note用の投稿文とアイキャッチ画像が30秒で生成されます

↓
3

確認・修正する

生成されたテキストは直接編集できます。「再生成」ボタンで何度でも作り直せます

↓
4

「投稿」ボタンを押すか、コピーして貼り付ける

APIキーを設定していればワンクリックで投稿。設定していない場合はコピーして各SNSに貼り付けます

FAQ
ターミナルを閉じると動かなくなります
bridge-server.mjs を起動しているターミナルを閉じると自動投稿ができなくなります。投稿文と画像の生成だけなら問題ありません。常時起動したい場合はlaunchdやpm2での自動起動設定をご検討ください。
画像が生成されません
Gemini APIの画像生成モデルは利用制限があります。設定タブで画像生成をONにした上で、しばらく待ってから再試行してください。
note投稿が失敗します
noteのログイン画面のデザインが変わると動作しなくなることがあります。GitHubの最新版にアップデートしてください（git pull）。
費用はかかりますか？
Gemini APIは無料枠で月数百回の生成ができます。X APIはBasicプラン（月100ドル）が必要です。Facebookとnoteは無料で使えます。投稿文のコピペだけであれば全て無料です。
